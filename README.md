# Doctor Appointment Booking Website

A simple frontend web application for booking doctor appointments.

## Features
- View Hospitals
- View Doctors
- Doctor Details Page
- Book Appointments
- User Login and Registration

## Tech Stack
- React.js
- JavaScript
- HTML
- CSS
- React Router DOM
- Vite

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Future Scope
- Backend using Node.js and Express.js
- MongoDB Database
- Authentication
- Online Payments
